piquid.pickleable module¶

class piquid.pickleable.PickleAble¶

Bases: object

Base class providing pickle serialization convenience methods.

This class provides convenience methods for pickle serialization. Serialization can either be to/from bytes or to/from a file path.

classmethod from_pickle_bytes(obj_bytes: bytes)¶

Deserialize and instance from bytes.

Parameters:

obj_bytes (bytes) – bytes containing a pickle serialized instance

Return type:

an instance of the class
