piquid.jsonable module

JSON serialization

class piquid.jsonable.JSONAble

Bases: object

A base class providing JSON serialization functionality.

This class includes an interface for JSON serialization that relies on the signature of the inheriting classes __init__ method. During serialization the parameters of the __init__ method are collected from the class to be serialized, and during deserialization the parameters of the __init__ method are used to create keyword arguments to the __init__ method. The means that any class members which need to be included in serialzied instances of the class must be included as paramters to the __init__ signature.

Notes & Caveats:

  • Currently only standard JSON types are supported.

  • Any instance state that is not part of the __init__ signature will not be serialized.
